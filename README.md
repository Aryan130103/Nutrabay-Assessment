# AI Resume Screener — Nutrabay

## Quick Start

1. Download `ai-resume-screener.html`
2. Open it in any browser (Chrome, Firefox, Edge)
3. Done — no install, no server, no setup required

---

## How to Use

1. Paste a Job Description or click **Load sample JD**
2. Upload a PDF resume or paste text for each candidate
3. Click **Analyze** and wait for results
4. Expand any candidate card for strengths, gaps and summary
5. Click **Generate Insights** for hiring recommendations
6. Click **Download Report** to export results

---

## Features

- AI scoring 0–100 per candidate
- Auto-ranking by score
- PDF resume upload + drag & drop support
- Strengths, gaps and fit recommendation per candidate
- AI hiring insights for the full batch
- One-click report download
- Built-in sample data for instant demo

---

## Tech

Vanilla HTML/CSS/JS · [Groq API](https://console.groq.com) (LLaMA 3.3 70B, free) · PDF.js
